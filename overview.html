PSBR II 2021 – Course overview

Philosophy behind the course structure

I took some inspiration from R for Data Science when it comes to what order to introduce different concepts. Their idea is to start with plotting, without any introduction to the R programming language whatsoever, including no explanation of the data types that are used when plotting. This is meant to start with introducing the visual stuff that you can play around with and get immediate (and sometimes pretty) feedback. After having seen what you can do in terms of visualising your data, only then do they start to talk about how to import data and tidying it up in order to be able to visualise it, using the visualisation as a motivation for the more boring task of cleaning you data.
